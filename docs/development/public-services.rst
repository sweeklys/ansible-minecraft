Used Public Services
=====================

For the Development we use a set of public services:

|Github Project Stars|
  `GitHub <https://github.com/nolte/ansible-minecraft>`_ for SoureCode and issue Tracking.

|Travis CI build status|
  `TravisCI <https://travis-ci.org/nolte/ansible-minecraft>`_ for simple tests execution an Ansible galaxy update notification.

|CircleCI build status|
  `CircleCI <https://circleci.com/>`_ for complex workflows, like releasing and publishing documentation.
